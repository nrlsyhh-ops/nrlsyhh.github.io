Project Overview

MUNCH is a food ordering and restaurant management website designed to make restaurant operations more organized. Customers can browse menus, add items to cart, place orders, make reservations and view order details. Staff can manage incoming orders and update order status, while the owner can monitor orders, manage menu items, manage staff accounts and view sales reports.

My Role

Contributed to UI/UX design, frontend development, database structure planning, PHP-MySQL connection, reservation functionality, staff ordering functions and system documentation such as ERD, UML diagrams and flowcharts.

Tools & Concepts

HTML CSS JavaScript PHP MySQL UI/UX Database Design System Architecture

System Architecture Summary

  • Presentation Layer: customer, staff and owner interfaces for login, menu browsing, ordering, order tracking and management.
  • Application Layer: customer modules, staff modules and owner/admin modules that handle cart, orders, status updates, menu management and reporting.
  • Database Layer: MySQL database for customer information, order information, sales information, menu information, staff information, receipt information and inventory information.
  • Payment Flow: payment gateway and transaction process connected to the ordering workflow.
