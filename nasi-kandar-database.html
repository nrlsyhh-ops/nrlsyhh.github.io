Database System · Group Project

Nasi Kandar Cik Yah — Food Ordering Database System

A database design and SQL project for a food ordering system. The project focused on organizing restaurant data, designing relationships between entities and writing SQL queries to support menu, order, sales and staff operations.

View Final Report Open Slides

Project Overview

This project was developed for the ICT200 course as a group presentation under the team name The Minions. The system represents a food ordering database for Nasi Kandar Cik Yah, including menu items, order records, staff handling, sales records and order-menu bridge data.

The main goal was to design a structured relational database and demonstrate useful SQL statements that retrieve significant business information from single and related tables.

My Role

Leader — contributed to planning, organizing the project flow, coordinating group work and preparing the presentation content.

Tools & Concepts

MySQL Database Design ERD SQL SELECT JOIN Aggregate Functions GROUP BY HAVING Subqueries

Database Scope

  • Menu & product data: Roti canai, lauk-lauk, add-ons, drinks and set menu pricing.
  • ERD entities: Orders, Sales, OrderMenu as a bridge entity, Menu and Staff.
  • SQL query coverage: Basic SELECT, comparison operators, LIKE, IN, JOIN, aggregate functions, GROUP BY, HAVING and subqueries.
  • Business information: menu price filtering, highest menu price by category, daily sales, total quantity sold and incomplete/unpaid order tracking.
