HyperHive — Personalized Adaptive Learning System

A system design project for Sekolah Sri Acmar that aims to improve students' study efficiency through personalized study planning, Pomodoro sessions, learning style preferences and progress tracking.

Project Overview

HyperHive, also known as the Personalized Adaptive Learning System (PALS), is a study management system designed to help students organize study sessions using the Pomodoro technique. The system supports user registration, login, subject management, study progress tracking and personalized learning preferences.

Purpose

The project addresses common student learning challenges such as distractions, poor time management, lack of motivation, unclear goals and difficulty understanding materials by proposing a structured learning support system.

My Role

Contributed to system analysis, module planning, flowchart design, pseudocode writing, expected screen preparation, documentation and presentation materials as part of the project team.

Main Features

  • User Authentication: register, login and forgot password modules.
  • Pomodoro Timer: structured study sessions and break duration planning.
  • Progress Tracking: monitors study sessions completed and learning progress.
  • Subject Management: add, delete and view study subjects.
  • User Preferences: learning style quiz and personalized study preferences.
